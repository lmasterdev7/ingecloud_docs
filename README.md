# Documentos del proyecto Ingecloud
En esta página se lista todos los documentos importantes que hacen parte del proyecto Ingecloud, el cual es una aplicación producida y distribuida por la empresa Ingecon Group SAS de la ciudad de Cali - Colombia. El objetivo es ordenar y hacer disponible documentos cómo políticas de seguridad, política de tratamiento de datos entre otros. De forma unificada se podrá acceder a los documentos necesarios.

También se agregan documentos relacionado con el despliegue y configuración de la aplicación para el departamento técnico de Ingecon Group SAS, como también los pormenores relacionados con la mantenimiento y la seguridad de la aplicación. Por cuánto este documento es público se omiten credenciales y contraseñas de acceso a la app, como también cualquier otro dato que pueda poner en riesgo la seguridad de Ingecon Group SAS o sus clientes.
