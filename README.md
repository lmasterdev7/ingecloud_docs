# Documentos del proyecto Ingecloud
En esta página se lista todos los documentos importantes que hacen parte del proyecto Ingecloud, el cual es una aplicación producida y distribuida por la empresa Ingecon Group SAS de la ciudad de Cali - Colombia. 

El objetivo es ordenar y hacer disponible documentos cómo políticas de seguridad, política de tratamiento de datos entre otros. De forma unificada se podrá acceder a los documentos necesarios.
